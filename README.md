# 4th Grade Math Quiz

An interactive math quiz for 4th graders, built as a single-page HTML app with canvas-based visuals.

**Live demo:** [mzrascoff.github.io/mathquiz](https://mzrascoff.github.io/mathquiz/)

## Topics

- **Multiplication** — Array/grid visuals with product calculations
- **Fractions** — Pie charts, bar models, comparing, and equivalent fractions
- **Geometry** — Perimeter, area, angle types, and polygon identification
- **Word Problems** — Division, money, addition, and difference problems

## Features

- 10 questions per quiz with multiple-choice answers
- Canvas-drawn visual aids for each question
- Progress bar and score tracking
- Instant feedback on correct/incorrect answers
- Results screen with performance message
- Mobile-responsive design

## Usage

No build step or dependencies required. Open `index.html` in a browser or visit the GitHub Pages deployment.
